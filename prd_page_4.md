8. Resource & Production System
8.1 Finite Resources
Each region contains capped extractable resources: - Oil - Wood - Metal - Gold
Extraction is rate-limited per cycle.
8.2 Production Chains
Raw resources must be processed through multiple businesses.
Examples:  - Oil → Refinery → Fuel → Gas Station → Transport - Metal → Machinery → Vehicles → Logistics -
Crops → Grocery → Consumers
This enforces interdependence and trade.
